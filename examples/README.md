# Examples

Interactive demos showing `VirtualList` in action.

## Run the demo

```bash
# From the repository root:
npx vite examples
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## What's included

| Demo | Description |
|------|-------------|
| **Simple List** | 10,000 plain strings with alternating row colors |
| **Contact List** | 50,000 typed objects with avatar + name + email |
| **Dynamic List** | Add / remove items on the fly |

Open DevTools → Elements to verify that only ~20-30 DOM nodes exist at any time, regardless of list size.
